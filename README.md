# Minimalist Luxury Business Card Website

This is a premium, responsive "Link-in-bio" website built with pure HTML, CSS, and Vanilla JavaScript. It is designed to be hosted for free on GitHub Pages.

## Project Structure
- `index.html`: The main page content.
- `style.css`: The styling (Minimalist Luxury theme).
- `script.js`: Interactive animations (fade-in, 3D tilt).
- `avatar.jpg`: Your profile photo (You need to add this).

## How to Customize

1.  **Photo:**
    - Place your photo in this folder and name it `avatar.jpg`.
    - If you don't add a photo, a placeholder with your initials will be shown automatically.

2.  **Links:**
    - Open `index.html` in any text editor (Notepad, VS Code).
    - Find the `href="..."` attributes and change the links if needed.

## key Features
- **Dark Mode Luxury Design:** Deep black background with gold accents.
- **Responsive:** Works perfectly on mobile phones and desktops.
- **Fast:** No heavy frameworks, loads instantly.
- **Interactive:** Subtle 3D tilt effect on buttons and smooth entry animations.

## How to Deploy to GitHub Pages

1.  **Create a Repository:**
    - Go to [GitHub.com](https://github.com) and create a new repository (e.g., `my-business-card`).
    - Make sure it is **Public**.

2.  **Upload Files:**
    - Click "Upload files" in your new repository.
    - Drag and drop `index.html`, `style.css`, `script.js`, and `avatar.jpg`.
    - Click "Commit changes".

3.  **Enable GitHub Pages:**
    - Go to **Settings** -> **Pages**.
    - Under **Source**, select `main` (or `master`) branch.
    - Click **Save**.
    - Wait a minute, and you will get your website link! (e.g., `yourusername.github.io/my-business-card`).
